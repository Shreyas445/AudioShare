# AudioShare
AudioShare is an android app that is used to share the audio of the phone to another phone Realtime in synchronization

the apps runs completely locally
we dont save any of your data in the cloud


turn the hotspot on in one phone and another phone with wifi connected to the hotsopt of it.
the person with wifi on can become host and start sharing the screen, the audio capture by the screen is shared to another phone
the person turning on hotspot can join the device and start hereing the audio


the app is in testing face, and is available for developer for testing
